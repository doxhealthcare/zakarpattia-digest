# Закарпаття — щоденний дайджест у Telegram

Серверний (хмарний) автопостинг топ-10 новин Закарпаття у канал **@zakarpattianews**
щодня о **19:00** (новини), о **09:00** (курс ПриватБанку + погода + бензин БРСМ) і по **пʼятницях ~17:00** (афіша на вихідні). Працює на GitHub Actions — без твого ПК і без щоденної ручної роботи.

Claude сам шукає новини (інструмент web_search), відбирає 10 найважливіших,
пише українською і публікує готовий пост. Один файл логіки — `digest.py`.

## Налаштування (одноразово, ~10 хв)

1. **Створи репозиторій** на GitHub (приватний підійде), залий у нього ці файли:
   - `digest.py` (новини 19:00), `morning.py` (курс+погода+бензин 09:00), `afisha.py` (афіша по пʼятницях)
   - `requirements.txt`
   - воркфлоу у `.github/workflows/`: `digest.yml`, `morning.yml`, `afisha.yml`

2. **Отримай ключ Anthropic API**: https://console.anthropic.com → API Keys → Create Key
   (це окремо від підписки Claude; поповни баланс на кілька доларів).

3. **Додай 2 секрети** у репозиторії: Settings → Secrets and variables → Actions → New repository secret:
   - `ANTHROPIC_API_KEY` — ключ із кроку 2
   - `TELEGRAM_BOT_TOKEN` — токен бота @zakarpattia_digest_bot

4. **Перевір зараз**: вкладка **Actions** → workflow «Zakarpattia digest» → **Run workflow**.
   Через ~1-2 хв у каналі має зʼявитися пост. Якщо щось не так — дивись лог запуску.

Далі нічого робити не треба: кожен день о 19:00 за Києвом пост виходить автоматично.

## Налаштування

- **Час**: змінюється у `.github/workflows/digest.yml` (cron у UTC; зараз 19:00 Київ) і в `digest.py` (`NOW.hour != 19`).
- **Модель / вартість**: за замовчуванням `claude-haiku-4-5` (найдешевша, ~$4-5/міс). Для кращої якості
  заміни у `env:` воркфлоу на `CLAUDE_MODEL: claude-sonnet-4-6` (~$8/міс) або `claude-opus-4-8` (~$12/міс).
- **Канал**: за замовчуванням `@zakarpattianews` (змінна `TELEGRAM_CHANNEL` у `digest.py`).

## Орієнтовна вартість

≈ **$4–5 / місяць** на Haiku (за замовчуванням), ~$8 на Sonnet, ~$12 на Opus
(плата за токени + інструмент веб-пошуку Anthropic API ≈$0.01/пошук). Сама інфраструктура GitHub Actions безкоштовна.
